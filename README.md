# Run

## Mongodb

- Change db dir path

` mongod --dbpath /path/to/dir `


- For viewing data
Download [Studio 3T or Robo 3T](https://robomongo.org/)


## Back-End

- ` cd backend `
- ` node main.js `

## Front-End

React and TypeScript

- ` cd my-app`
- ` npm run start `
